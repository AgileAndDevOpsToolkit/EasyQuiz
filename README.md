# EasyQuiz

EasyQuiz est une application web de Quiz simple d'utilisation, conçue pour animer des ice-breakers lors de meetings agiles.

## Fonctionnalités

- Création rapide d'un quiz
- Interface légère utilisée par l'animateur du Quiz
- Scoreboard

## Prérequis

- Disposer d'un navigateur web

## Installation rapide

- Simplement télécharger la page index.html et l'ouvrir dans un navigateur web
